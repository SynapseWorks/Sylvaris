# Ecology Systems

## Core Philosophy

The world of Sylvaris is alive and reactive.

Ecosystems:
- evolve
- recover
- collapse
- migrate
- retaliate

The world continues changing even without player involvement.

## World State Layers

### 1. Biome Health
Tracks ecological stability.

### 2. Creature Ecology
Tracks migration, aggression, and population behavior.

### 3. Faction Pressure
Tracks extraction, warfare, industrialization, and expansion.

### 4. Mythic Instability
Tracks resonance imbalance and ecosystem retaliation.

## Biome Health Variables

Suggested variables:
- Resonance Stability
- Corruption Level
- Biodiversity
- Extraction Pressure
- Mythic Activity
- Climate Stability

## Dynamic Ecosystem Effects

Healthy ecosystems:
- vibrant wildlife
- stable weather
- peaceful resonance
- creature abundance

Strained ecosystems:
- migration changes
- instability
- creature aggression
- resource scarcity

Collapsed ecosystems:
- corruption spread
- silence
- mythic retaliation
- environmental disasters

## Creature Migration

Creatures migrate dynamically due to:
- storms
- corruption
- industrialization
- ecological collapse
- seasonal cycles

## Mythic Retaliation

If ecosystems become unstable enough:
- mythic creatures awaken
- weather escalates
- ecosystems retaliate violently

Wildbound groups may intentionally trigger these events.

## Corruption

Corruption behaves like:
- resonance fragmentation
- ecological imbalance
- spiritual instability

Effects:
- hallucinations
- biome distortion
- creature instability
- silence
- traversal hazards

## Dynamic Events

Suggested event types:
- extraction convoys
- migration crises
- corruption blooms
- mythic awakenings
- storms
- political unrest
- sabotage attacks
- ecosystem collapse

## Event Escalation Structure

Events evolve through stages:
- Rumor
- Emerging
- Crisis
- Escalation
- Resolution

Ignoring events still changes the world.

## Information Systems

Players learn about events through:
- rumors
- radio
- travelers
- creature behavior
- faction broadcasts
- resonance visions
- environmental changes

## Ecosystem Recovery

Players may:
- restore rivers
- reconnect migration paths
- stabilize resonance
- heal corruption
- calm mythic creatures

Recovery is never fully permanent.

## Time Progression

The world operates on:
- seasons
- weather cycles
- migration periods
- political developments
- ecological shifts

The game uses soft time pressure rather than constant urgency.
