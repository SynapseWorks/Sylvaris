# Sylvaris

Sylvaris is an open-world solarpunk creature-bond action RPG set in a spiritually alive world where humanity’s relationship with nature has fractured.

Players explore a dynamic continent shaped by ecological systems, creature relationships, political factions, and mythic resonance power.

Core themes:
- coexistence vs domination
- inherited responsibility
- ecological balance
- spiritual connection
- the ethics of power

Current Status:
Pre-production / prototype planning.
